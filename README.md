# Coinbase-S: HoneyCoin Crypto Integration Dashboard

![HoneyCoin](https://via.placeholder.com/1200x600/8b5cf6/ffffff?text=HoneyCoin+Dashboard) <!-- Replace with screenshot -->

A stunning, interactive financial model for **HoneyCoin** (Nairobi fintech) with **live Coinbase crypto feeds**. Track revenue, EBITDA, users, and crypto impacts in real-time.

## Features
- **Live Data**: USD/KES FX + BTC/ETH prices from Coinbase API.
- **Scenarios**: Best/Base/Downside projections (2026-2028).
- **Expansion**: Kenya → East Africa → Pan-Africa modeling.
- **Charts**: Recharts for revenue growth & user trends (crypto-adjusted).
- **UI**: Glassmorphism, dark mode, neon animations.

## Quick Start
```bash
pnpm install
pnpm dev  # Runs on http://localhost:3000
